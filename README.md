# UMHackathon-KreditX
AI-powered credit scoring and loan decision platform for unbanked rural micro-entrepreneurs.\

[KreditXPitchingVideo](https://drive.google.com/file/d/1MqCzs-HrWMe_mWy-tMGyo1RHqW7n5Gsb/view?usp=sharing)

# Tech Stack:
•Frontend: Streamlit\
•Backend: FastAPI + Python \
•AI: Google Gemini API\
•Database: SQLite

# Getting started: 
```bash
Frontend
pip install streamlit
```
# Running the app: 
```bash
Backend
uvicorn kreditX:app --reload --port 9000

In a new terminal, start the Streamlit frontend
streamlit run frontend.py
```
# Key Features:
•Supports natural chats in Bahasa Malaysia or English.\
•Organises mixed data from WhatsApp chats, voice notes, and scanned paper notes into structured credit profiles.\
•Generates explainable credit scores, recommended microloan amounts, and stakeholder suggestions in simple language.

# Monetization model: 
•Pay per use\
